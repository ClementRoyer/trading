## Run

```
$ git clone git@github.com:ClementRoyer/trading.git
$ cd trading

------ Option One ------
- Run with custom file -
$ cd push_index
$ ./run

------ Option Two ------
- Run with online data -
$ cd trade_online
$ ./command.sh

```

## Edit

You can edit the file `command.sh` in root folder, `push_index/run` will use this file to start.
